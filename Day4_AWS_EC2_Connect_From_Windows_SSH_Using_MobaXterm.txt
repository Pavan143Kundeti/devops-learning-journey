# 1. First Understand What SSH Actually Is

**SSH (Secure Shell)** is a network protocol.

Definition in simple words:

SSH allows your computer to **securely control another computer over the internet**.

Example situation:

You create an EC2 server in AWS.
That server is running Linux somewhere in an AWS data center.

To control it, you must log in remotely.

SSH creates that secure connection.

---

# 2. What Happens Technically

Your laptop sends a request like this:

```
ssh -i key.pem ubuntu@3.89.45.22
```

This means:

* use SSH
* use the private key
* login as user `ubuntu`
* connect to server IP

Once connected, your terminal becomes the **server terminal**.

---

# 3. Where Can You Run This SSH Command?

Any program that supports SSH can run it.

Examples:

| Tool             | Type                |
| ---------------- | ------------------- |
| Command Prompt   | Windows terminal    |
| PowerShell       | Windows terminal    |
| Git Bash         | Linux-like terminal |
| Windows Terminal | Modern terminal     |
| VS Code terminal | Developer terminal  |
| MobaXterm        | SSH client          |

All of these **do the same job**.

They simply send SSH requests to the server.

---

# 4. Why the Instructor Uses MobaXterm

Because **many Windows beginners struggle with SSH setup**.

Problems beginners face:

1. Windows command prompt historically did not support SSH
2. Users don't know how to configure keys
3. File transfer becomes difficult
4. Linux commands feel uncomfortable

So tools like MobaXterm simplify everything.

---

# 5. What MobaXterm Actually Does

MobaXterm is a **GUI SSH client**.

Instead of typing commands manually, you fill fields.

Example inside MobaXterm:

You enter:

Host

```
3.89.45.22
```

Username

```
ubuntu
```

Private key

```
windows-demo.pem
```

Then click **Connect**.

Behind the scenes, MobaXterm runs the same command:

```
ssh -i windows-demo.pem ubuntu@3.89.45.22
```

So nothing special is happening.

It is just easier for beginners.
