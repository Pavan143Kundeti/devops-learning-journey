# 2. Script to List All Processes

Interview question:

> Write a script to list all processes.

Command used:

```bash
ps -ef
```

Meaning

```
ps → process status
-e → show all processes
-f → full details
```

Example output

```
UID   PID  PPID CMD
root  1     0   systemd
user  221   1   bash
```

---

## Follow-up Question

Interviewer might ask:

> Only print process IDs.

Example output line:

```
root 1234 1 0 bash
```

Column breakdown

```
1 → user
2 → process ID
3 → parent process
```

They want **column 2**.

---

### Solution using awk

```bash
ps -ef | awk '{print $2}'
```

Explanation

```
ps -ef → get processes
| → pipe output
awk → filter column
$2 → column 2
```

# 3. Print Only Errors From Remote Logs

Real DevOps scenario:

Logs may be stored in:

* AWS S3
* remote server
* log storage

You fetch logs using **curl**.

---

## Command Example

```bash
curl google.com
```

Returns webpage content.

---

## Filter specific lines

Use:

```bash
grep
```

Example

```bash
curl google.com | grep href
```

This prints only lines containing **href**.

---

## Real log example

If log file has:

```
INFO service started
TRACE connecting database
ERROR connection failed
```

To get only errors:

```bash
curl logfile_url | grep ERROR
```

---

## Why pipe is required

Pipe:

```
|
```

Means:

Send output of command 1 → into command 2.

Without pipe

```
curl logfile grep error
```

System thinks **grep is hostname**.

Pipe connects them.

---

## Command Breakdown

```
curl → download remote file
grep → filter text
| → connect commands
```

# 4. Numbers Divisible by 3 and 5 but Not 15

Interview logic question.

Example numbers:

```
1 → 100
```

Numbers divisible by 3

```
3
6
9
12
15
```

Numbers divisible by 5

```
5
10
15
```

But we must **exclude 15**.

---

## Script

```bash
#!/bin/bash

for i in {1..100}
do
  if [[ ($i%3 -eq 0 || $i%5 -eq 0) && $i%15 -ne 0 ]]
  then
     echo $i
  fi
done
```

---

## Logic Explanation

```
%i → modulus operator
%3 → divisible by 3
%5 → divisible by 5
%15 → divisible by 15
```

Condition

```
(3 OR 5) AND NOT 15
```

# 5. Count Number of "s" in a Word

Example word

```
Mississippi
```

Count how many **s**.

Answer

```
4
```

---

## Script

```bash
#!/bin/bash

x="Mississippi"

grep -o "s" <<< "$x" | wc -l
```

---

## Explanation

```
grep -o → print each match separately
<<< → pass variable as input
wc -l → count lines
```

So each **s** becomes a line.

Then wc counts lines.

# 7. What is CronTab?

CronTab = **task scheduler in Linux**

It runs commands **automatically at a scheduled time**.

Example:

Run script every day at **6 PM**.

Instead of manually logging in.

---

## Example Cron Entry

```bash
0 18 * * * /home/script.sh
```

Meaning

```
0 → minute
18 → hour (6pm)
* → every day
```

---

Real DevOps use cases

```
backup databases
send reports
rotate logs
health monitoring
run scripts
```

# 9. Difference Between Soft Link and Hard Link

Important Linux concept.

Command used

```
ln
```

---

# Hard Link

Hard link = **actual copy reference of file**

Example

```
fileA
fileB
```

Both point to **same data block**.

Even if original file deleted

Hard link still works.

---

# Soft Link (Symbolic Link)

Soft link = **shortcut to file**

Example

```
python → python3
```

If original deleted

Soft link **breaks**.

---

## Command

Create soft link

```bash
ln -s fileA linkA
```

## Break

