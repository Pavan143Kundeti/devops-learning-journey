# DevOps Shell Scripting – Day 7

## Project: AWS Resource Usage Tracker

### Goal

Create a **shell script that tracks AWS resource usage** and generates a report automatically.

Resources tracked:

* EC2 Instances
* S3 Buckets
* Lambda Functions
* IAM Users

The script will run **automatically using Cron Job**.

---

# 1. Why Companies Move to Cloud

Main reasons:

### 1. Manageability

If a company maintains its own servers:

* Needs data center
* Needs hardware maintenance
* Needs patching
* Needs system engineers

Cloud providers handle this.

Example providers:

* Amazon Web Services
* Microsoft Azure

---

### 2. Cost Efficiency

Cloud uses **Pay As You Go model**.

Example problem:

Developer creates:

* 100 EC2 instances
* Many EBS volumes

But they are **not used**.

Still you pay for them.

So DevOps engineers must **monitor resource usage**.

**Shell Script + AWS CLI**

# 15. Problem with Raw Output

AWS CLI returns **large JSON output**.

Example:

```bash
aws ec2 describe-instances
```

Output contains many fields.

But we only need:

```
InstanceId
```

---

# 16. Using JSON Parser (jq)

Use tool:

```
jq
```

Purpose:

Parse JSON output.

Example command:

```bash
aws ec2 describe-instances | jq '.Reservations[].Instances[].InstanceId'
```

Explanation:

| Part         | Meaning          |
| ------------ | ---------------- |
| Reservations | Top level object |
| Instances    | Instance list    |
| InstanceId   | Specific field   |

Output becomes clean:

```
i-02345abc
i-98765xyz
```

---

# 17. Updated EC2 Command

```bash
echo "Print list of EC2 instances"

aws ec2 describe-instances \
| jq '.Reservations[].Instances[].InstanceId'
```

```
>> append output to file
```

File becomes daily report.

# 19. Automate Script with Cron Job

Cron allows **scheduled execution**.

Edit cron:

```bash
crontab -e
```

Example schedule:

```
0 18 * * * /home/ec2-user/aws_resource_tracker.sh
```

Meaning:

```
Minute Hour Day Month DayOfWeek
```

```
0 18 = 6 PM everyday
```

So script runs **daily automatically**.
